# TGA Offer Page - Black Oaks Land Clearing

This project is a high-ticket contractor lead generation page for Black Oaks Land Clearing (BOLC), built with a vanilla HTML/CSS/JS stack.

## Project Specifications
- **Client:** Black Oaks Land Clearing (BOLC)
- **Niche:** Land Clearing, Excavation, Forestry Mulching
- **Aesthetic:** Gold and Navy (TGAMA-Project standard)
  - Gold: `#d4af37`
  - Navy: `#001f3f`
- **Breakpoints:** 768px for mobile responsiveness
- **Primary Goal:** Lead generation via a multi-step quiz

## Technical Details
- `index.html`: Main landing page structure
- `style.css`: Custom styling using CSS variables for consistent branding
- `script.js`: Dynamic quiz logic and form handling
- `base_site.html`: Legacy/reference crawl from Hardscape Contractors site

## Maintenance
- Ensure all modifications adhere to the gold-and-navy aesthetic.
- Maintain a focus on high-ticket contractors ($250K+ revenue target).
- All changes should be pushed to the `master` branch.
